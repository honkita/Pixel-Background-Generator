# Pixel-Background-Generator
## What is this
This is an ongoing work in progress application that can generate dithered pixel backgrounds. I created this originally to make a pixel background for my website. ![default](https://github.com/honkita/Pixel-Background-Generator/assets/57504343/2a33bad4-ba02-48b1-82ce-9bcbb7562758)
## What additional features can you expect from this in the foreseeable future?
I am planning on adding the presets I have for my website along with size of canvas along with grid scale (1 pixel can be 2x2, etc.). Finally, I am planning on fixing the display issues and provide easier user viewing. 
## How many colours does this support?
Current, this supports only four colours. The ability to have different number of colours may be implemented in the future. This is not a feature that is currently being worked on.
